    The differences between Web1, Web2, Web3:

    Web 1.0 Web 2.0 Web 3.0
    Mostly Read-Only Wildly Read-Write Portable and Personal
    only few content creators with huge majority of users who are passive consumers of content. Used as participative social web. People publish content that other people can consume Create and share content through search and analysis, based on the capability to interpret information and use it to draw conclusions
    There is no technology that can perform a search, we have to search for the requested information ourselves. Suitable for static sites like articles. It is possible to query sites and get all the content that exists on the topic, but not necessarily the answer The network strives to find the best information, not only to show information but also process, analyze and understand it

    Use case example:

    Query: "Washing Machine - BOSCH"

    - Web 1.0: Displays read-only technical information about the washing machine.
    - Web 2.0: Displays product reviews written by people and places where you can buy the product.
    - Web 3.0: Displays shopping sites based on Artificial Intelligence and recommendations for similar products.
